# Fluxo legado de certificado

O certificado é gerado no resultado, inclui participante, e-mail mascarado, emissor, pesquisa, data/hora, pontuação/faixa, código de validação e URL pública `index.html#certificate/validate/{code}`. O botão Baixar/Imprimir usa o pipeline de PDF/print existente sem expor token completo.
