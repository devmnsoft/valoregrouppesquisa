# Painel Operacional

O Valora.Web ASP.NET MVC expõe `/Operations` com Bootstrap, JavaScript, jQuery e AJAX consumindo Valora.Api via HTTP. Exibe API, banco, SMTP, fila, e-mails pendentes/falhos, último envio, última falha, pesquisa gratuita, certificado público, CSP, Firebase legado e Valora.Web sem segredos.
