# Arquitetura Backend Valora Pulse
Solução em `backend/Valora.sln`: `Valora.Api` (controllers/JWT/Swagger), `Valora.Application` (DTOs, contratos, serviços), `Valora.Domain` (entidades futuras), `Valora.Infrastructure` (Dapper, PostgreSQL, segurança, auditoria) e `Valora.Tests`.
A Fase 1 é paralela ao Firebase e não substitui o frontend atual.
