# Checklist UX Real

- Login com e-mail e senha.
- Cadastro de empresa com LGPD.
- Forgot/reset com mensagens seguras.
- Dashboard consome `/me`, `/plans/public`, `/health`.
- Planos consomem `/plans/public`.
- Pesquisa pública valida e envia respostas.
- Resultado usa token temporário em sessionStorage.
- Certificados usam `requestBinary` com fallback JSON.
- Status consulta health endpoints.
- Mobile reforçado.
