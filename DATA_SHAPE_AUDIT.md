# Auditoria de formatos de dados

Execute:

```bash
node scripts/audit-data-shapes.js --project gestordepesquisa
```

O relatório JSON é gravado em `reports/data-shape-audit.json` e verifica `settings.faq`, planos, módulos, empresas, organizações, usuários, formulários, pesquisas, respostas, base de conhecimento e suporte.
