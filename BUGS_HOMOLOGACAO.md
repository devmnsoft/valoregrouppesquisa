# Bugs de Homologação — Valora Pulse

**Versão:** Valora Group™ 8.6.4 RC1
**Data de abertura:** 2026-06-21

## Critérios de severidade

### Bloqueante

Classificar como **bloqueante** quando o bug:

- impede login;
- impede cadastro;
- impede resposta;
- gera cálculo errado;
- causa vazamento entre empresas;
- gera erro crítico em produção;
- quebra tela principal;
- representa falha grave de segurança.

### Alta

Classificar como **alta** quando:

- fluxo importante falha;
- relatório sai errado;
- e-mail não envia;
- dashboard fica incoerente.

### Média

Classificar como **média** quando envolver:

- problema de UX;
- ajuste de layout;
- mensagem confusa.

### Baixa ou melhoria

Classificar como **baixa** ou **melhoria** quando envolver:

- refinamento visual;
- texto;
- pequena melhoria sem impacto funcional relevante.
