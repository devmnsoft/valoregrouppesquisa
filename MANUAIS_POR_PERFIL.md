# Manuais por perfil

O serviço `manual-service.js` centraliza `PROFILE_MANUALS` para público, Admin Valora, Consultor Valora, Empresa Admin, Gestor de Pesquisa, Analista de Resultados, Gestor de Área, Participante e Convidado Externo.

Cada manual contém título, descrição, primeiros passos, seções, rotas relacionadas, FAQ, resumo de permissões e próximos passos. A interface global abre **Manual do meu perfil** para usuário logado e **Ajuda e perguntas frequentes** para público.

## Teste rápido
1. Entrar com cada credencial demo.
2. Clicar no botão global do manual.
3. Confirmar título e permissões do perfil.
4. Usar botões de rota, ValoraBot e Falar com atendente.
