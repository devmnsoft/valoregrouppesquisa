# Endpoints API
- `GET /health`, `GET /health/database`
- `POST /admin/database/migrate` somente Development/Local
- `POST /auth/register-company`, `POST /auth/login`, `POST /auth/forgot-password`, `GET /me`
- `GET /plans/public`, `GET /plans/{id}`
- `GET /organizations/current`, `PATCH /organizations/current`, `GET /organizations/current/usage`
- `GET /surveys/public/{token}`, `POST /surveys/{surveyId}/responses`, `GET /responses/{responseId}/result`
