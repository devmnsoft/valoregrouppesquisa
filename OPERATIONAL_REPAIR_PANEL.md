# Painel Admin de Operação Assistida
O menu Admin > Operação Assistida contém cartões de Status Geral, Config Produção, Firebase Blaze, Cloud Functions, SMTP/E-mail, Pesquisa Gratuita, Fila de E-mail, Menu Mobile, Erros Recentes, Custos/Quotas, Backup/Rollback e LGPD/Auditoria.

Cada cartão informa status ok/warn/error, última verificação, mensagem curta, ação recomendada e botão de revalidar. Ações sensíveis exigem admin_valora. O painel não exibe SMTP_PASSWORD, senha, token completo, publicToken completo, resultToken completo, tokenHash, private_key, service account ou API key completa.
