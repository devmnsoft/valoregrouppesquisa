# Manuais por perfil

Os manuais por perfil ficam em `manual-service.js` e são usados tanto pelo botão **Manual do meu perfil** quanto pelo ValoraBot.

## Perfis cobertos

- Público / Visitante.
- Admin Valora.
- Consultor Valora.
- Administrador da Empresa.
- Gestor de Pesquisa.
- Analista de Resultados.
- Gestor de Área.
- Participante.
- Convidado Externo.

## Estrutura do manual

Cada manual contém título, descrição, primeiros passos, seções, rotas relacionadas, perguntas frequentes, resumo de permissões e próximos passos. A interface global abre **Manual do meu perfil** para usuário logado e **Ajuda e perguntas frequentes** para público.

O ValoraBot adiciona a ação **Abrir manual** quando uma resposta pode ser aprofundada pelo manual do perfil.

## Teste rápido

1. Entrar com cada credencial demo.
2. Clicar no botão global do manual.
3. Confirmar título e permissões do perfil.
4. Usar botões de rota, ValoraBot e Falar com atendente.
